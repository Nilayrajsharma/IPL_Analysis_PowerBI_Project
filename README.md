Overall Aim of the Project

The aim of this project is to analyze Indian Premier League (IPL) data using interactive Power BI dashboards to uncover insights related to team performance, player statistics, match outcomes, and venue trends, enabling data-driven understanding of tournament patterns and performance factors.


Dashboard 1 – Tournament Overview Analysis
Aim

To provide a high-level overview of the IPL tournament by analyzing total matches, seasons, teams, and overall match outcomes. This dashboard helps users quickly understand the scale and structure of the IPL across different seasons.

Focus:

Total seasons & matches

Teams participation

Overall match results


Dashboard 2 – Team Performance Analysis
Aim

To analyze team-wise performance across IPL seasons by evaluating matches played, wins, losses, and winning percentages. This dashboard helps identify the most consistent and successful teams in IPL history.

Focus:

Team-wise wins & losses

Performance comparison

Dominant teams analysis


Dashboard 3 – Player Performance Analysis
Aim

To evaluate individual player performance by analyzing runs scored, wickets taken, strike rate, and consistency. This dashboard helps identify top-performing batsmen, bowlers, and all-rounders.

Focus:

Top run scorers

Leading wicket takers

Player contribution analysis


Dashboard 4 – Match & Venue Analysis
Aim

To analyze match outcomes across different venues and conditions. This dashboard helps understand how location, toss decisions, and venue characteristics impact match results.

Focus:

Venue-wise matches

Toss impact on results

Home vs away performance


Dashboard 5 – Season-wise Trend Analysis
Aim

To analyze season-wise trends in IPL by comparing team performance, scoring patterns, and competitiveness across different years. This dashboard highlights how the IPL has evolved over time.

Focus:

Season-wise performance trends

Changes in scoring patterns

Team dominance over years
